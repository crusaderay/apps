Learn the basic stuff

Learn the basic stuff

1. Learn how to write md file

  • tutorial website
  • 标题: #, 可以多层
  • 段落：两个空格或者直接空一行是换行
  • 斜体：* xxx *
  • 粗体：** xxx **
  • 粗斜体：*** xxx ***
  • 分割线： ***
  • 删除线：~~
  • 下划线：<u> xxx </u>
  • 注脚： [^ xxx ], [^ xxx ]: explain
  • 无序列表：+, -, * 后面加空格
  • 有序列表：1. 2. 后面加空格
  • 列表嵌套：前面加四个空格即可
  • 特殊区块：> 可以嵌套其他列表或者叠加，如果列表嵌套区块，则需要加四个空格，反之不需要。
  • 代码：` xxx ` 或者是 整段 ``` xxxxxxx ```
  • 链接：[xxx](http) or < http > or [1], [1]: http
  • 图片：![alt 属性文本](图片地址) 或者 ![alt 属性文本](图片地址 "可选标题") 也可以用连接的方式添加图片(最后一种方式)
  • 表格：
    • Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行
    • -: 设置内容和标题栏居右对齐。:- 设置内容和标题栏居左对齐。:-: 设置内容和标题栏居中对齐。
  • 其他：(这些用的应该不多，可以参考链接)
    • 可以使用大部分的HTML元素
    • 可以画流程图
    • 可以写数学公式

2. Other stuff

  • github page链接文件要取决于当前链接已经在的那个文件夹，比如已经在articles下了，再链接就不需要在写文件夹，直接括号链接即可
    • 例如：xxx.com/articles/ 在调用同文件夹下的文件时就不用写（/articles/xxx）
  • 根目录标识： ../
